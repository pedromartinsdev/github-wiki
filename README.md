# Github Wiki

## :books: Projeto

Este projeto é uma wiki de repositórios do Github feito com React.

Os dados são consumidos da API do Github, [Github API](https://api.github.com/).

![preview](src/assets/preview.gif)

## :computer: Tecnologias

- React
- Styled components
- Axios
